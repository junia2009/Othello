# Deep Othello 🌊

深海をテーマにしたオセロ（リバーシ）の PWA ゲームです。

**[→ プレイする](https://junia2009.github.io/Othello/)**

---

## 特徴

### ゲームプレイ
- **CPU対戦 / 2人対戦** の2モード
- CPU難易度: 弱（Easy）/ 中（Normal）/ 強（Hard）/ 鬼（Oni・隠し）
  - Hard はミニマックス法＋アルファベータ枝刈り（探索深度 4）
  - **鬼（イースターエッグ）**: 強に5回勝つと解放。探索深度6＋着手順序付け、終盤10マス以下は完全読み切り、評価関数も強化（角の確定石・フロンティア石・機動力を加味）
- **石の色選択**: CPU戦で闇（先手）または光（後手）を選べる
- ヒント表示トグル（打てる場所を光るオーブで表示）

### 深海ビジュアル
- クラゲ、魚、珊瑚、海藻などの深海生物アニメーション
- 生物発光グロースポット・光線シャフト・コースティクス
- 石を置いた時のパーティクルバースト
- フリップ時の連鎖エフェクト（置いた石からの距離に応じた時差アニメーション）
- スコア +N フロートテキスト

### 画面遷移
- タイトル ↔ 対戦画面の切り替え時に深海ダイブ演出（泡が上昇しながらフェード）

### PWA 対応
- ホーム画面への追加（iOS / Android）
- オフライン動作（Service Worker によるキャッシュ）
- 自動サイレント更新（新バージョン検出次第、メニュー画面で自動リロード）

---

## ファイル構成

```
index.html      ゲーム本体（HTML / CSS / JS 完全単一ファイル）
sw.js           Service Worker（Cache-First + 自動更新）
manifest.json   PWA マニフェスト
icon.svg        アイコン（深海・オセロモチーフ）
icon-192.png    PWA アイコン 192px
icon-512.png    PWA アイコン 512px
```

## リリース手順

新バージョンをリリースする際は以下の2箇所を同時に更新してください。

```js
// index.html
const APP_VERSION = 'vX.Y.Z';

// sw.js
const VERSION = 'vX.Y.Z';
```

バージョンが変わると Service Worker のキャッシュキーが変わり、既存ユーザーに更新が自動配信されます。
